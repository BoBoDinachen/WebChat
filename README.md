# WebChat
>* web上面的聊天应用
>* 🍉作者: XDEcat
>* 包含前端和服务端
>* ✨前端技术栈：JavaSript、WebSocket、React、Sass、socket.io-client
>* ✨后端技术栈: Node、Express、Nodejs-websocket、socket.io
>* ✨数据库: MongoDB
